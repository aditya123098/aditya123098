### 📄 PDF Summarizer
> **AI-powered document summarization using RAG**

An intelligent tool that reads any PDF and generates concise, accurate summaries using LLMs and retrieval-augmented generation.

**How it works:**
- 📥 Upload any PDF document
- ✂️ Chunks text using smart splitter
- 🔢 Embeds chunks into vector store (FAISS)
- 🤖 LLM generates context-aware summary
- 📊 Extracts key insights & highlights

**Tech Stack:**
`Python` `LangChain` `PyPDF2` `FAISS` `OpenAI` `Streamlit`

### ❓ Document Q&A System
> **Ask questions over any document using Advanced RAG**

A conversational AI system that answers natural language questions from documents with accurate, cited responses.

**How it works:**
- 📂 Load documents (PDF, TXT, DOCX)
- 🔍 Semantic search via vector embeddings
- 🔄 Re-ranks & retrieves top-k chunks
- 💬 LLM answers with context & citations
- 🗂️ Full chat history support

**Tech Stack:**
`Python` `LangChain` `LlamaIndex` `ChromaDB` `OpenAI` `Streamlit`
